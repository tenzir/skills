# Status

Describes status of a finding.

- **Full name**: `google.backstory.Status`
- **Values**: `5`

## Values

### `STATUS_UNSPECIFIED`

- **Number**: `0`

Unspecified finding status.

### `NEW`

- **Number**: `1`

New finding.

### `REVIEWED`

- **Number**: `2`

When a finding has feedback.

### `CLOSED`

- **Number**: `3`

When an analyst closes an finding.

### `OPEN`

- **Number**: `4`

Open. Used to indicate that a Case / Alert is open.
