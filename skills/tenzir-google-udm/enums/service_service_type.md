# Service.ServiceType

The type of service.

## Values

0. `SERVICE_TYPE_UNSPECIFIED`: Default service type.
1. `KERNEL_DRIVER`: A kernel driver.
2. `FILE_SYSTEM_DRIVER`: A file system driver.
3. `WIN32_OWN_PROCESS`: A process that is owned by the service. This is a Windows-specific service type.
4. `WIN32_SHARE_PROCESS`: A process that is shared by the service. This is a Windows-specific service type.
5. `ADAPTER`: An adapter. This is a Windows-specific service type.
6. `RECOGNIZER_DRIVER`: A recognizer driver. This is a Windows-specific service type.
7. `INTERACTIVE_PROCESS`: An interactive process. This is a Windows-specific service type.
