# Authentication.Outcome

The outcome of the authentication event.

- **Full name**: `google.backstory.Authentication.Outcome`
- **Values**: `3`

## Values

### `OUTCOME_UNSPECIFIED`

- **Number**: `0`

The default outcome.

### `SUCCESS`

- **Number**: `1`

The authentication was successful.

### `FAILURE`

- **Number**: `2`

The authentication failed.
