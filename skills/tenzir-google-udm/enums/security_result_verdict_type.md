# SecurityResult.VerdictType

Category of the verdict.

## Values

0. `VERDICT_TYPE_UNSPECIFIED`: Verdict category not specified.
1. `PROVIDER_ML_VERDICT`: MLVerdict result provided from threat providers, like Mandiant. These fields are used to model Mandiant sources.
2. `ANALYST_VERDICT`: Verdict provided by the human analyst. These fields are used to model Mandiant sources.
