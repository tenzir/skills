# Browser.Cookie.CookieSameSite

The SameSite attribute of a cookie.

- **Full name**: `google.backstory.Browser.Cookie.CookieSameSite`
- **Values**: `4`

## Values

### `COOKIE_SAME_SITE_UNSPECIFIED`

- **Number**: `0`

Default value.

### `STRICT`

- **Number**: `1`

Corresponds to SameSite=Strict.

### `LAX`

- **Number**: `2`

Corresponds to SameSite=Lax.

### `NONE`

- **Number**: `3`

Corresponds to SameSite=None.
