# User.AccountType

User Account Type.

## Values

0. `ACCOUNT_TYPE_UNSPECIFIED`: Default user account type.
1. `DOMAIN_ACCOUNT_TYPE`: A human account part of some domain in directory services.
2. `LOCAL_ACCOUNT_TYPE`: A local machine account.
3. `CLOUD_ACCOUNT_TYPE`: A SaaS service account type (such as Slack or GitHub).
4. `SERVICE_ACCOUNT_TYPE`: A non-human account for data access.
5. `DEFAULT_ACCOUNT_TYPE`: A system built in default account.
