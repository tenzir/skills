# Browser.UrlVisitType

The type of visit to a URL.

## Values

0. `URL_VISIT_TYPE_UNSPECIFIED`: Default value.
1. `LINK`: The user clicked a link.
2. `TYPED`: The user typed a URL.
3. `AUTO_BOOKMARK`: The user bookmarked the URL.
4. `AUTO_SUBFRAME`: Loaded in a nested subframe by the parent frame.
5. `MANUAL_SUBFRAME`: Loaded in a nested subframe by the user.
6. `GENERATED`: The user clicked on auto generated link in browser address bar.
7. `AUTO_TOPLEVEL`: The page was loaded through command line or is the starting page.
8. `FORM_SUBMIT`: The user submitted a form.
9. `RELOAD`: The user reloaded the page.
10. `KEYWORD`: The Url was generated by a keyword search configured by user.
11. `KEYWORD_GENERATED`: Corresponds to a visit generated by a keyword search.
12. `REDIRECT`: The user was redirected to the URL.
