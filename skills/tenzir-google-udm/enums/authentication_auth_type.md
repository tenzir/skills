# Authentication.AuthType

Type of system the authentication event is associated with.

## Values

0. `AUTHTYPE_UNSPECIFIED`: The default type.
1. `MACHINE`: A machine authentication.
2. `SSO`: An SSO authentication.
3. `VPN`: A VPN authentication.
4. `PHYSICAL`: A Physical authentication (e.g. "Badge reader").
5. `TACACS`: A TACACS family protocol for networked systems authentication (e.g. TACACS, TACACS+).
