- Type: `string` (singular)

### `format`

- Type: `string` (singular)

### `compilationTime`

- Type: `google.protobuf.Timestamp` (singular)

Code sign timestamp

### `teamId`

- Type: `string` (singular)

The assigned team identifier of the developer who signed the application.
