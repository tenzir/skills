# microseconds

> Converts a number to equivalent microseconds.

Converts a number to equivalent microseconds.

```tql
microseconds(x:number) -> duration
```

## Description

This function returns microseconds equivalent to a number, i.e., `number * 1us`.

### `x: number`

The number to convert.
