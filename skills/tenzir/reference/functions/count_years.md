# count_years

> Counts the number of years in a duration.

Counts the number of `years` in a duration.

```tql
count_years(x:duration) -> float
```

## Description

This function returns the number of years in a duration, i.e., `duration / 1y`.

### `x: duration`

The duration to count in.
