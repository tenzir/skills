# nanoseconds

> Converts a number to equivalent nanoseconds.

Converts a number to equivalent nanoseconds.

```tql
nanoseconds(x:number) -> duration
```

## Description

This function returns nanoseconds equivalent to a number, i.e., `number * 1ns`.

### `x: number`

The number to convert.
