# ocsf::class_uid

> Returns the classuid for a given classname.

Returns the `class_uid` for a given `class_name`.

```tql
ocsf::class_uid(name:string) -> int
```

## Description

### `name: string`

The `class_name` for which `class_uid` should be returned.
