# ocsf::category_name

> Returns the categoryname for a given categoryuid.

Returns the `category_name` for a given `category_uid`.

```tql
ocsf::category_uid(id:int) -> string
```

## Description

### `id: int`

The `category_uid` for which `category_name` should be returned.
