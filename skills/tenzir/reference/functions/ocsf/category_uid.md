# ocsf::category_uid

> Returns the categoryuid for a given categoryname.

Returns the `category_uid` for a given `category_name`.

```tql
ocsf::category_uid(name:string) -> int
```

## Description

### `name: string`

The `category_name` for which `category_uid` should be returned.
