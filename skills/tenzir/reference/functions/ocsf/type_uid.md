# ocsf::type_uid

> Returns the typeuid for a given typename.

Returns the `type_uid` for a given `type_name`.

```tql
ocsf::type_uid(name:string) -> int
```

## Description

### `name: string`

The `type_name` for which `type_uid` should be returned.
