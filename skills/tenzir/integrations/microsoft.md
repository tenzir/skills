# Microsoft integration

> Connect Azure, Microsoft 365, and Windows: stream through Event Hubs, store in Blob Storage, ingest into Sentinel, and parse the full Windows telemetry surface.

Tenzir connects the Microsoft estate end to end: stream events through [Azure Event Hubs](microsoft/azure-event-hubs.md), store and replay them in [Blob Storage](microsoft/azure-blob-storage.md), send telemetry to [Sentinel and Log Analytics](microsoft/sentinel-log-analytics.md), and collect signals from [Defender](microsoft/defender.md), [Microsoft Graph](microsoft/graph.md), and [Windows Event Logs](microsoft/windows-event-logs.md) — all from the same pipeline language.

The [Microsoft package](https://app.tenzir.com/library) is the deepest in the Tenzir Library: 70+ operators that parse and map Windows, Active Directory, Azure, Microsoft 365, and Defender telemetry to OCSF.
