# Changelog


Welcome to the Tenzir changelog hub. Here you can find release notes, feature updates, and behind-the-scenes improvements across our projects.

[View all changes chronologically](changelog/timeline/2026.md)

[RSS Feed](/changelog/feed.xml)

[ Tenzir Node v5.28.0 ](changelog/tenzir.md)

[The security data pipeline engine that collects security telemetry, normalizes events to OCSF, and offers an open data lake for storage.](changelog/tenzir.md)

[ Tenzir Platform v1.29.0 ](changelog/platform.md)

[The control plane for managing Tenzir Nodes, providing a web interface for data exploration and pipeline creation, and workspace and secret management.](changelog/platform.md)

[ Tenzir MCP Server v0.4.5 ](changelog/mcp.md)

[A Model Context Protocol (MCP) server that connects AI agents to Tenzir, generates data parsers, and creates OCSF mappings.](changelog/mcp.md)

[ Tenzir Test v1.7.3 ](changelog/tenzir-test.md)

[An integration testing framework that discovers tests, runs them with managed fixtures, and validates output against baselines.](changelog/tenzir-test.md)

[ Tenzir Ship v1.3.1 ](changelog/tenzir-ship.md)

[Ship faster with automated release engineering. Manage changelogs, generate release notes, publish GitHub releases.](changelog/tenzir-ship.md)

[ Claude Marketplace v3.0.0 ](changelog/claude-plugins.md)

[A Claude Code plugin marketplace containing plugins that extend Claude's capabilities for working with the Tenzir ecosystem.](changelog/claude-plugins.md)

[ Tree-sitter TQL v1.0.0 ](changelog/tree-sitter-tql.md)

[A Tree-sitter grammar for the Tenzir Query Language (TQL) with bindings for Node.js, Python, Rust, Go, and Swift.](changelog/tree-sitter-tql.md)

***

For general release announcements and deeper dives into selected features, check out our [blog](https://tenzir.com/blog) or join the conversation on our [Discord](https://tenzir.com/discord).
