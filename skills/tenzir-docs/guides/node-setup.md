# Overview


The [Tenzir Node](../explanations/node.md) is the vehicle to run [pipelines](../explanations/pipeline.md). It is light-weight server application that can be deployed on-premises or in the cloud.

Setting up a node involves the following basic steps:

1. [Provisioning](node-setup/provision-a-node.md): Create a new node in the [platform](../explanations/platform.md) and obtain a token.
2. [Sizing](node-setup/size-a-node.md): Determine the hardware requirements for the node based on the expected workload.
3. [Deploying](node-setup/deploy-a-node.md): Deploy the node to a server or cloud provider.
4. [Configuring](node-setup/configure-a-node.md): Tweak the default settings to optimize performance and security.
