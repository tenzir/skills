# count_weeks

> Counts the number of weeks in a duration.

Counts the number of `weeks` in a duration.

```tql
count_weeks(x:duration) -> float
```

## Description

This function returns the number of weeks in a duration, i.e., `duration / 1w`.

### `x: duration`

The duration to count in.
