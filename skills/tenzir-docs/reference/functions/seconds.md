# seconds

> Converts a number to equivalent seconds.

Converts a number to equivalent seconds.

```tql
seconds(x:number) -> duration
```

## Description

This function returns seconds equivalent to a number, i.e., `number * 1s`.

### `x: number`

The number to convert.
