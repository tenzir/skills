# count_milliseconds

> Counts the number of milliseconds in a duration.

Counts the number of `milliseconds` in a duration.

```tql
count_milliseconds(x:duration) -> float
```

## Description

This function returns the number of milliseconds in a duration, i.e., `duration / 1ms`.

### `x: duration`

The duration to count in.
