# weeks

> Converts a number to equivalent weeks.

Converts a number to equivalent weeks.

```tql
weeks(x:number) -> duration
```

## Description

This function returns weeks equivalent to a number, i.e., `number * 1w`.

### `x: number`

The number to convert.
