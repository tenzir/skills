# Metric

> The Metric object defines a simple name/value pair entity for a metric.


The Metric object defines a simple name/value pair entity for a metric.

## Attributes

**`name`**

* **Type**: `string_t`
* **Requirement**: required

The name of the metric.

**`value`**

* **Type**: `string_t`
* **Requirement**: required

The value of the metric.
