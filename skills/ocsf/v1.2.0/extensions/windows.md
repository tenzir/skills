# Windows

The Windows extension defines Windows specific attributes, objects and classes.

- **Name**: `win`
- **UID**: `2`

## Events

- `prefetch_query` - Prefetch Query
- `registry_key` - Registry Key Activity
- `registry_key_query` - Registry Key Query
- `registry_value` - Registry Value Activity
- `registry_value_query` - Registry Value Query
- `resource` - Windows Resource Activity

## Objects

- `registry_key` - Registry Key
- `registry_value` - Registry Value
- `win_resource` - Windows Resource
