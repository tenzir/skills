# Windows

The Windows extension defines Windows specific attributes, objects and classes.

- **Name**: `win`
- **UID**: `2`

## Events

- `prefetch_query` - Prefetch Query
- `registry_key_activity` - Registry Key Activity
- `registry_key_query` - Registry Key Query
- `registry_value_activity` - Registry Value Activity
- `registry_value_query` - Registry Value Query
- `windows_resource_activity` - Windows Resource Activity
- `windows_service_activity` - Windows Service Activity

## Objects

- `process` - Process
- `query_evidence` - Query Evidence
- `registry_key` - Registry Key
- `registry_value` - Registry Value
- `startup_item` - Startup Item
- `evidences` - Windows Evidence Artifacts
- `win_resource` - Windows Resource
- `win_service` - Windows Service
