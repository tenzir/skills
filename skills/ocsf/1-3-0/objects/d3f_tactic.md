## Attributes

### `name`

- **Type**: `string_t`
- **Requirement**: optional

The tactic name that is associated with the defensive technique, as defined by D3FENDTM Matrix. For example: `Isolate`.

- **Type**: `url_t`
- **Requirement**: optional
